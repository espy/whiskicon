Whiskicon

A small mod of @lipka's Piecon by espy

  • Whiskicon.setProgress(Number)
    to set progress
  • Whiskicon.setState(String)
    to set state, accepts "play", "pause" and null
  • Whiskicon.setOptions({
  background: '#F3F68D',    // Background color
  ring: '#FFF',             // Ring background color
  percentage: '#3C3D40',    // Ring foreground color
  icons: '#3C3D40',         // Icon color
  thickness: 1,             // Ring thickness
  fallback: false           // Use title fallback
});
